# CardLauncherPro
一款为 Android 设备开发的桌面，高效、简洁、快速、流畅、美观是最大的特点。大量使用了 RecycleView,CardView 改善交互体验，并遵循 Google Material Design。
## 注
此项目为个人摸索学习作品，其中不少组件来自各位开源作者，但没有标明从何处引用了何人的代码，在此对参与开源、代码、技术分享的各位作者表示敬意和感谢！
